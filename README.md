# *runner-unity-ecs*
A reference implementation of a simple runner game, using Unity ECS

## **Unity Dependencies**
- Unity ECS
- Hybrid Render Pipeline

## **Custom Dependencies*
- Based on my [Unity Game Template](https://github.com/SMaleck/unity-game-facilities), which uses
    - UniRX
    - Zenject
    - DOTween
